// Geospatial Fraud Risk Estimation

This project implements a geospatial fraud risk scoring pipeline based on historical inspection data.

The system estimates fraud probability using hierarchical spatial aggregation across multiple geographic levels.

// Features

- Geospatial fraud detection
- Hierarchical geohash aggregation
- Scalable parquet data processing
- Fraud risk scoring pipeline

// Technologies

- Python
- Pandas
- NumPy
- Geohash
- Parquet

// Pipeline

1. Load inspection data
2. Load geospatial reference data
3. Compute fraud frequency by geohash
4. Estimate fraud probability
5. Export fraud risk scores

// Note

Due to confidentiality restrictions, the original datasets are not included.
Synthetic data can be used to reproduce the pipeline.
